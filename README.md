# GTG Perfumes — Assignment 3

A luxury perfume brand landing page built with **HTML**, **CSS**, and **Vanilla JavaScript** as part of a UI/Frontend assignment.

## 🌐 Live Preview

Open `index.html` in your browser to view the project locally.

## 📁 Project Structure

```
assigment 3/
├── index.html      # Main HTML page
├── styles.css      # All styles and responsive layout
├── script.js       # Interactive functionality
├── *.jpg / *.png   # Product and lifestyle images
└── README.md       # Project documentation
```

## ✨ Features

- **Hero Section** — Full-screen banner with stats (10x power, 9 ingredients, 20K+ customers)
- **Product Gallery** — Image carousel with thumbnail grid and dot navigation
- **Subscription Plans** — Single & Double subscription options with fragrance selector
- **Fragrance Collection** — Accordion-style collection showcase (Signature, Citrus, Floral, Oud)
- **Stats Strip** — Animated percentage-based user satisfaction stats
- **Blog Table** — Article listing with read time and categories
- **Comparison Table** — GTG vs competitors (Arose, Bella, Daisies)
- **Newsletter Form** — Email subscription with validation
- **Responsive Design** — Mobile hamburger menu, fluid layouts
- **Scroll Animations** — Reveal-on-scroll for all sections

## 🛠️ Tech Stack

| Technology | Usage |
